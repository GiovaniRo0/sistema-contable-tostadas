Sistema Contable para Local de Tostadas

Sistema web para gestión de ventas, gastos y balances de un negocio de alimentos.

Tecnologías
- HTML5
- PHP
- PostgreSQL
- Bootstrap

Funcionalidades
- Registro de ventas diarias
- Control de gastos
- Generación de balances
- Reportes por período

Instalación
1. Instalar PostgreSQL y crear base de datos
2. Configurar credenciales 
3. Desplegar en servidor web con soporte PHP
